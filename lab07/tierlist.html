Reflection

This project was a lot of fun. I was able to pick a topic I enjoy to make a table on which made it even more entertaining. The semantics of formatting the table and forms I found to be somewhat frustrating. I finished the form and table relatively quickly but the formatting was off and it took me a long time using youtube and W3 to find solutions so that everything was aesthetically pleasing. The radio buttons and check box were also a hassle to format correctly, but overall I am happy with the way my project turned out!
