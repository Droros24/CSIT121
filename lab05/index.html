Lab 5

What I've Learned

This week I learned about link anchors and media queries. I found it to be very insightful as we have all come across a website that may be desktop friendly but very difficult to navigate on mobile devices. I've learned that mobile devices have no surpassed desktops when it comes to web traffic. According to Kinsta mobile devices are now accounting for over 51% of web traffic. This being said, it is very important to make sure your websites are not only desktop friendly but also mobile and table friendly to keep up with current demands. I learned the min and max width of px for each device as setting @media to a max 600px will be used to design the mobile experience, between 601px and 900px can be used to design tablet experiences and anything from 901px and greater will help design the desktop experience.

Best Design Practices

When designing a website it is best to think of how you want to organize it prior to entering all the content. To start every page is made of rows, you should conside the layout you would like before beginning in rows and columns. These rows backgrounds should not be the same twice. This will help differentiate the rows and content within them. You should also have the content you need before you layout the website. Similar to wordpressing, information hierachry is key when design a well organized website. Having the content before creating the layout will let you organzie the information properly so you can design the website based off key content.
